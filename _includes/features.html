Built for teams

LightTag makes annotating with a team simple. As a project manager you define how many annotators you'd like to work on each example.
LightTag will automatically allocate work accordingly, and let you aggregate annotations or view them on an annotator by annotator basis. You'll never need to assign work yourself or track which examples have been covered, LightTag does it for you.
